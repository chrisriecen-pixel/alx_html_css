# Headphones Landing Page

This project is a responsive HTML/CSS landing page built from scratch using semantic markup and accessibility best practices. It replicates a Figma design without any external libraries or JavaScript.

## Features

- Responsive layout for desktop, tablet, and mobile
- Semantic HTML structure
- Accessibility-friendly form and navigation
- Custom hover/active styles
- Max content width: 1000px centered

## Notes

- No external CSS frameworks used
- No JavaScript used
- Fonts: Source Sans Pro, Spin-Cycle-OT (fallbacks applied if unavailable)
